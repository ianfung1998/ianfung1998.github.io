By Ian

发表于 2015-10-09

二、注册GitHub账号、购买域名

  1. 在GitHub按步骤一步步注册账号，并创建repository,详细步骤请参考如何搭建一个独立博客

    • 在创建博客之前，当然必须有GitHub的帐号，该帐号将用来创建项目，默认的域名username.github.com/projectName中的username也要用到这个帐号。

    • 访问：GitHub，注册你的username和邮箱，邮箱十分重要，GitHub上很多通知都是通过邮箱的。比如你的主页上传并构建成功会通过邮箱通知，更重要的是，如果构建失败的话也会在邮件中说明原因。

    • 在创建博客之前，还需要用已有的帐号创建一个项目，上面那个链接的projectName将是这里即将创建的项目名称。在Git中，项目被称为仓库(Repository)，仓库顾名思义，当然可以包含代码或者非代码。将来我们的网页或者模板实际上都是保存在这个仓库中的。

    • 创建了仓库后，我们就需要管理它，无论是管理本地仓库还是远程仓库都需要Git客户端。Git客户端实际上十分强大，它本身就可以offline的创建本地仓库，而本地仓库和远程仓库之间的同步也是通过Git客户端完成的。

    • 这里省略了windows下安装和使用Git客户端的基本技巧，您应该已经掌握此技能了。虽然，您仍然可以按照本教程的指引完成一个简单的网站，但是后期的维护工作无论如何都不能少了这项技能。

  2. (可选，有域名更好一些，好记，好推广)购买域名推荐在godaddy上购买，可以避免好多麻烦，什么备&案之类的，也可以用支付宝，比较方便。不过使用了一些网上搜的优惠码，支付宝有不能用的情况。我直接使用的是官方的优惠，没有使用搜到的码。

    • 按照步骤购买域名，并设置

三、配置

主目录配置文件_config.yml

在步骤一第3条中初始化好的blog文件夹中有一个名为_config.yml的配置文件，直接把我的配置文件贴上来，供大家参考
注意：配置文件中每个:号后都有一个空格 

# Hexo Configuration  
## Docs: http://hexo.io/docs/configuration.html  
## Source: https://github.com/hexojs/hexo/  

# Site  
title: Ian’s Blog  
subtitle: Yeah  
description: No  
author: Ian  
language: zh-CN  
timezone:  

# URL  
## If your site is put in a subdirectory, set url as 'http://yoursite.com/child' and root as '/child/'  
url: http://china-developer.com  
root: /  
permalink: :year/:month/:day/:title/  
permalink_defaults:  

# Directory  
source_dir: source  
public_dir: public  
tag_dir: tags  
archive_dir: archives  
category_dir: categories  
code_dir: downloads/code  
i18n_dir: :lang  
skip_render:  

# Writing  
new_post_name: :title.md # File name of new posts  
default_layout: post  
titlecase: false # Transform title into titlecase  
external_link: true # Open external links in new tab  
filename_case: 0  
render_drafts: false  
post_asset_folder: false  
relative_link: false  
future: true  
highlight:  
      enable: true  
      line_number: true  
      auto_detect: true  
      tab_replace:  

# Category & Tag  
default_category: uncategorized  
category_map:  
tag_map:  

# Date / Time format  
## Hexo uses Moment.js to parse and display date  
## You can customize the date format as defined in  
## http://momentjs.com/docs/#/displaying/format/  
date_format: YYYY-MM-DD  
time_format: HH:mm:ss  

# Pagination  
## Set per_page to 0 to disable pagination  
per_page: 10  
pagination_dir: page  

# Extensions  
## Plugins: http://hexo.io/plugins/  
## Themes: http://hexo.io/themes/  
theme: jacman  

# Deployment  
## Docs: http://hexo.io/docs/deployment.html  
deploy:  
      type: git  
  repository: https://github.com/ianfung1998/ianfung1998.github.io.git  
  branch: master  

# 建站日期  
since: 2015  
# 头像(使用七牛云存储做图床)  
avatar: http://7xn3um.com1.z0.glb.clouddn.com/face.jpg  
# 社交 （社交网站个人主页）  
social:  
      github: https://github.com/ianfung1998  
# 多说 （在多说官网注册后，添加站点，绑定域名，找到自己的duoshuo_shortname）  
duoshuo_shortname: china-developer  
#百度统计 （在百度统计注册后，添加站点，填写id）  
baidu_analytics: e08519*************a464  
# JiaThis 分享服务  
jiathis: true  
# 百度分享服务  
baidushare: false  
# 多说分享服务  
duoshuo_share: false  
# Swiftype Search Key  
# swiftype_key: zhMqCY6*******voNv  
# 多说热评文章 true 或者 false
